ABout this project

The fine-tuned DistilBERT model accurately classifies SMS messages as "Spam" or "Not Spam," delivering fast, real-time predictions through a web app. This system helps users filter unwanted messages and guard against spam or phishing. It also serves as a foundation for more advanced solutions to manage and secure text-based communication in the digital age..
